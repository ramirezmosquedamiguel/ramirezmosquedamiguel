## Featured projects

### 360° Ultrasonic Obstacle Detection System

Embedded prototype that combines two HC-SR04 ultrasonic sensors, one servomotor, Arduino firmware, serial communication and a real-time Python graphical interface.

The sensors are mounted back to back to provide a combined 360-degree representation of detected obstacles, including invalid-measurement handling and distance-based visualization.

[View project](https://github.com/ramirezmosquedamiguel/360-obstacle-detection-radar)

### Driver Drowsiness Monitor

Computer vision prototype for real-time driver eye-closure detection using Python, OpenCV, MediaPipe and the Eye Aspect Ratio method.

The system implements continuous eye-closure timing, a two-stage non-blocking audio alarm and a simulated vehicle safety response.

[View project](https://github.com/ramirezmosquedamiguel/driver-drowsiness-monitor)
